# 5.3 LOT 및 시리얼 번호 체계

[← 목차로 돌아가기](../../README.md)

> **참고:** LOT 및 시리얼 번호 체계의 상세 내용은 [2.6 LOT 및 시리얼 번호 체계](../02-product-process.md#26-lot-및-시리얼-번호-체계)를 참조하세요.

## 5.3.1 LOT 번호 형식

**형식:** `{제품모델코드}-{공장코드}-{YYMMDD}{교대}-{일련번호}`

**예시:** `PSA10-KR-251110D-001`

| 요소 | 설명 | 예시 |
|------|------|------|
| 제품모델코드 | 제품 모델 코드 (가변 길이) | PSA10 |
| 공장코드 | 생산 공장 | KR |
| YYMMDD | 생산 날짜 | 251110 |
| 교대 | Day/Night | D |
| 일련번호 | 당일 순번 | 001 |


## 5.3.2 시리얼 번호 형식

**형식:** `{LOT번호}-{순번}`

**예시:** `PSA10-KR-251110D-001-0001`

- LOT당 0001~0100 (100개)
- Label Printing 공정에서 순차 생성
