## 1. 개요

### 1.1 WIP 시스템이란?

**WIP (Work In Progress)**: 작업 중인 제품을 추적하기 위한 임시 식별 번호 시스템

**주요 특징**:
- 공정 1에서 WIP ID 생성
- 공정 2~6에서 WIP ID로 추적
- 공정 7에서 Serial Number로 전환
- 공정 8에서 Serial Number로 완료

### 1.2 WIP ID vs Serial Number

| 구분 | WIP ID | Serial Number |
|------|--------|---------------|
| **생성** | 공정 1 (레이저 마킹) | 공정 7 (라벨 프린팅) |
| **사용** | 공정 1~6 | 공정 7~8 |
| **형식** | `WIP-KR01PSA2511-001` | `KR01PSA25110001` |

## 2. 관리자 가이드

### 2.1 LOT 생성 및 관리

#### 2.1.1 LOT 생성

**단계**:

1. MES 시스템 로그인
2. **LOT 관리** 메뉴 선택
3. **새 LOT 생성** 버튼 클릭
4. 정보 입력:
   - 제품 모델: PSA10, WFO 등 선택
   - 목표 수량: 100 (기본값)
   - 교대조: D (주간) / N (야간)
5. **생성** 버튼 클릭

**결과**:
- LOT 번호 자동 생성: `KR01PSA2511001`
- LOT 상태: `CREATED`

#### 2.1.2 WIP 생성 시작

**조건**:
- LOT 상태가 `CREATED`일 것
- 공정 1 작업자가 준비되었을 것

**단계**:

1. **LOT 상세** 페이지에서 **WIP 생성 시작** 버튼 클릭
2. 작업자 선택
3. 레이저 마킹 장비 선택
4. **확인** 클릭

**결과**:
- LOT 상태: `CREATED` → `IN_PROGRESS`
- 공정 1 작업 시작 가능

## 3. 작업자 가이드 - 공정 1 (WIP 생성)

### 3.1 작업 준비

**필요 장비**:
- 레이저 마킹 기계
- 바코드 리더기
- 태블릿 (MES 앱)

**준비 사항**:
1. 레이저 마킹 기계 전원 ON
2. MES 앱 로그인
3. LOT 번호 확인

### 3.2 WIP 생성 절차

#### 단계 1: LOT 선택

1. MES 앱 실행
2. **공정 1 - 레이저 마킹** 메뉴 선택
3. 진행 중인 LOT 목록에서 작업할 LOT 선택
4. **WIP 생성** 버튼 탭

#### 단계 2: WIP 착공

```
화면:
┌─────────────────────────────────────────┐
│ 공정 1: 레이저 마킹                      │
├─────────────────────────────────────────┤
│ LOT: KR01PSA2511001                     │
│ 진행: 0/100                              │
├─────────────────────────────────────────┤
│                                          │
│   [  다음 WIP 생성  ]                   │
│                                          │
└─────────────────────────────────────────┘
```

**동작**:
1. **다음 WIP 생성** 버튼 탭
2. 시스템이 자동으로 WIP ID 생성
3. 화면에 WIP ID 표시: `WIP-KR01PSA2511-001`

#### 단계 3: 레이저 마킹 작업

1. LMA 케이스를 레이저 마킹 기계에 장착
2. 자동으로 WIP ID가 레이저로 각인됨
3. 약 60초 소요

#### 단계 4: 바코드 검증

1. 마킹된 바코드를 리더기로 스캔
2. 시스템이 자동으로 검증
   - ✅ **성공**: WIP ID 일치 확인
   - ❌ **실패**: 마킹 불량, 재작업 필요

#### 단계 5: 완공 처리

**성공 시**:
```
화면:
┌─────────────────────────────────────────┐
│ ✅ 공정 1 완료                           │
├─────────────────────────────────────────┤
│ WIP ID: WIP-KR01PSA2511-001            │
│                                          │
│ 다음 공정: 공정 2 (LMA 조립)             │
│                                          │
│   [  완공  ]   [  불량 처리  ]           │
│                                          │
└─────────────────────────────────────────┘
```

1. **완공** 버튼 탭
2. WIP 상태: `CREATED` → `IN_PROGRESS`
3. 제품을 다음 공정으로 이동

**불량 시**:
1. **불량 처리** 버튼 탭
2. 불량 유형 선택
3. 해당 LMA 폐기

### 3.3 작업 팁

✅ **DO**:
- 바코드 검증 필수 수행
- WIP ID가 명확하게 각인되었는지 육안 확인
- 작업 완료 후 즉시 완공 처리

❌ **DON'T**:
- 바코드 검증 생략 금지
- 불량 제품을 다음 공정으로 이동 금지
- WIP ID를 수동으로 기록하지 말 것 (자동 생성)

## 4. 작업자 가이드 - 공정 2~6 (WIP 작업)

### 4.1 작업 준비

**필요 장비**:
- 바코드 스캐너
- 태블릿 (MES 앱)
- 공정별 작업 도구 및 측정 장비

### 4.2 WIP 스캔 및 착공

#### 단계 1: WIP 스캔

```
화면:
┌─────────────────────────────────────────┐
│ 공정 2: LMA 조립                         │
├─────────────────────────────────────────┤
│                                          │
│   [  WIP 바코드 스캔  ]                 │
│                                          │
│   또는 수동 입력:                        │
│   ________________________              │
│                                          │
└─────────────────────────────────────────┘
```

1. 이전 공정에서 전달받은 제품의 바코드 스캔
2. WIP ID 자동 인식: `WIP-KR01PSA2511-001`
3. 시스템이 이전 공정 완료 확인

#### 단계 2: 착공 확인

```
화면:
┌─────────────────────────────────────────┐
│ ✅ WIP ID 확인 완료                      │
├─────────────────────────────────────────┤
│ WIP: WIP-KR01PSA2511-001               │
│ LOT: KR01PSA2511001                     │
│ 순번: 1/100                              │
├─────────────────────────────────────────┤
│ 이전 공정:                               │
│ ✅ 공정 1: 레이저 마킹 (PASS)           │
├─────────────────────────────────────────┤
│ 다음 공정:                               │
│ 공정 2: LMA 조립                         │
│                                          │
│   [  착공  ]                            │
│                                          │
└─────────────────────────────────────────┘
```

1. WIP 정보 확인
2. **착공** 버튼 탭
3. 착공 시간 자동 기록

### 4.3 작업 수행

**공정 2 (LMA 조립)** 예시:
1. SMA 스프링, 모선 케이블 조립
2. 소요 시간: 약 60분
3. 육안 검사 수행

**공정 3 (센서 검사)** 예시:
1. 온도 센서 검사 (60℃ ±1℃)
2. TOF 센서 검사 (I2C 통신)
3. 자동 측정 및 판정
4. 소요 시간: 약 60초

### 4.4 완공 처리

#### 성공 시 (PASS)

```
화면:
┌─────────────────────────────────────────┐
│ 공정 2 작업 완료                         │
├─────────────────────────────────────────┤
│ WIP: WIP-KR01PSA2511-001               │
│                                          │
│ 작업 시간: 58분                          │
│                                          │
│ 결과 선택:                               │
│   ( ) PASS - 합격                       │
│   ( ) FAIL - 불합격                     │
│                                          │
│ 비고: _______________________           │
│                                          │
│   [  완공  ]                            │
│                                          │
└─────────────────────────────────────────┘
```

1. **PASS** 선택
2. 필요시 비고 입력
3. **완공** 버튼 탭
4. 제품을 다음 공정으로 이동

#### 불량 시 (FAIL)

1. **FAIL** 선택
2. 불량 유형 선택:
   - 부품 불량
   - 조립 불량
   - 센서 이상
   - 기타
3. 불량 상세 내용 입력
4. **완공** 버튼 탭
5. 재작업 또는 폐기 대기

### 4.5 공정별 특이사항

#### 공정 3 (센서 검사)

**자동 측정**:
- 시스템이 자동으로 온도/TOF 센서 측정
- 합격 기준 자동 판정
- 측정 데이터 자동 저장

**재검사**:
- 1차 실패 시 즉시 재검사 가능
- **재검사** 버튼으로 재시도

#### 공정 4 (펌웨어 업로드)

**업로드 실패 시**:
- 자동 재시도 (최대 3회)
- 3회 실패 시 작업자 알림
- 케이블 연결 확인 후 수동 재시도

#### 공정 6 (성능검사)

**측정 항목**:
- 온도 (℃)
- 변위 (mm)
- 힘 (kgf)

**측정 데이터 자동 저장**:
- 합격 범위: 목표값 ±2kgf
- 측정 데이터는 자동으로 시스템에 기록

### 5.3 문제 해결

#### 공정 1~6 미완료 오류

```
화면:
┌─────────────────────────────────────────┐
│ ❌ Serial 전환 불가                      │
├─────────────────────────────────────────┤
│ WIP ID: WIP-KR01PSA2511-001            │
│                                          │
│ 완료되지 않은 공정:                      │
│ ❌ 공정 5: 로봇 조립 (미완료)           │
│                                          │
│ 공정 5를 먼저 완료해주세요.              │
│                                          │
└─────────────────────────────────────────┘
```

**조치**:
1. 해당 공정으로 제품 이동
2. 공정 완료 후 재시도

#### 프린터 오류

**용지 부족**:
- 라벨 용지 보충
- MES 앱에서 **재시도** 버튼 탭

**프린터 연결 끊김**:
- 프린터 전원 확인
- USB/네트워크 연결 확인
- 프린터 재시작 후 재시도

## 8. 문제 해결 가이드

### 8.1 일반적인 문제

#### 문제 1: WIP ID 스캔이 안 됨

**원인**:
- 바코드 손상
- 스캐너 오류
- 조명 불량

**해결**:
1. 바코드 육안 확인 (긁힘, 오염)
2. 스캐너 청소
3. 조명 밝게 조정
4. 수동 입력 시도

#### 문제 2: 이전 공정 미완료 오류

```
오류 메시지:
❌ 이전 공정이 완료되지 않았습니다
현재 공정: 공정 3
완료 필요: 공정 2
```

**해결**:
1. 해당 제품을 이전 공정으로 되돌림
2. 이전 공정 완료 후 재시도

#### 문제 3: Serial 전환 실패

**원인**:
- 공정 1~6 중 일부 미완료
- 일부 공정 FAIL 상태

**해결**:
1. WIP 상세 정보에서 미완료 공정 확인
2. 해당 공정 완료 또는 재작업
3. 모든 공정 PASS 후 재시도

### 8.2 장비 문제

#### 레이저 마킹 기계 오류

**증상**: 마킹 불량, 바코드 판독 실패

**해결**:
1. 레이저 초점 조정
2. LMA 케이스 위치 확인
3. 레이저 출력 확인
4. 장비 재시작

#### 프린터 오류

**증상**: 라벨 출력 실패

**해결**:
1. 용지 확인 및 보충
2. 프린터 연결 확인
3. 프린터 큐 초기화
4. 프린터 재시작

### 8.3 시스템 오류

#### MES 앱 오류

**증상**: 앱 응답 없음, 로딩 지연

**해결**:
1. 네트워크 연결 확인
2. 앱 재시작
3. 태블릿 재시작
4. IT 팀에 문의

#### API 오류

**증상**: 서버 응답 없음, 타임아웃

**해결**:
1. 네트워크 상태 확인
2. 서버 상태 확인 (관리자)
3. 잠시 후 재시도
4. 긴급 시 수동 기록 후 사후 입력
