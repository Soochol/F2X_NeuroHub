## 1. 프로젝트 개요

### 1.1 프로젝트 명
**F2X NeuroHub MES (Manufacturing Execution System) 구축**

### 1.2 프로젝트 목적
- 생산 공정 데이터 실시간 수집 및 관리
- LOT 및 시리얼 번호 기반 이력 추적
- 생산 현황 실시간 모니터링 대시보드
- 품질 데이터 수집 및 분석

### 1.3 주요 기능
- LOT 관리 (생성, 착공, 완공)
- 8개 공정 관리 (레이저 마킹 ~ 포장)
- 시리얼 번호 자동 발급
- 라벨 프린팅 자동화
- 펌웨어 배포 및 버전 관리
- 실시간 대시보드 (생산 현황, 품질 데이터)
- 외부 공정 앱 연동 (JSON 파일 기반)

### 1.4 기술 스택
- **Backend:** FastAPI (Python 3.11+), PostgreSQL 15+, Redis 7.0+
- **Frontend:** PyQt5 (작업 PC), React 18 (관리자 대시보드)
- **Infrastructure:** Ubuntu 22.04 LTS, Docker, Nginx
- **Monitoring:** Prometheus + Grafana

### 1.5 규모
- 동시 접속자: 100명
- 작업 PC: 7-20대
- 일일 생산량: 100대
- 공정 수: 8개
